ATIRADOR DE MONTANHA

Mountain Shooter é um jogo digital em 2D criado com o uso da biblioteca Pygame, trazendo uma proposta simples, porém dinâmica, voltada para entretenimento rápido e interação entre jogadores. A ideia principal do jogo é proporcionar uma experiência inspirada em títulos clássicos, com foco em ação e desafios que exigem atenção e reflexos.

O projeto também permite que dois jogadores participem juntos, tornando a experiência mais interativa e colaborativa. Além disso, conta com um sistema de registro de pontuação, no qual os resultados das partidas podem ser armazenados para acompanhamento posterior.

Esse jogo foi desenvolvido como parte das atividades do curso de Análise e Desenvolvimento de Sistemas, na disciplina de Linguagens de Programação Aplicadas. Durante sua criação, foram explorados conceitos importantes da programação em Python, bem como noções iniciais relacionadas ao desenvolvimento de jogos.
